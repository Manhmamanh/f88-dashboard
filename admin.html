Trang quản trị Dashboard

Cập nhật dữ liệu, link và tiêu đề — bấm Lưu là tự đăng lên hệ thống, dashboard cập nhật sau ~1 phút. · Mở dashboard ↗

1. Kết nối GitHub chưa kết nối

Dán mã Token GitHub để có quyền cập nhật. Chỉ cần nhập 1 lần, trình duyệt sẽ nhớ.

Chưa có token? Tạo tại github.com/settings/tokens → tick quyền repo → Generate token → copy dán vào đây. Token là chìa khoá riêng của bạn, đừng chia sẻ cho người khác.

2. Cập nhật dữ liệu (Excel)

Tải file Excel mới (cấu trúc cột giống file gốc). Hệ thống sẽ thay toàn bộ dữ liệu dashboard.

📊 Bấm để chọn file Excel (.xlsx) hoặc kéo thả vào đây

3. Tiêu đề & Link chi tiết

Chỉnh tiêu đề trên đầu dashboard và link "Xem dữ liệu chi tiết".
